Summarative Reflection

Prior to this course I had taken some far more basic web development courses in high school. I came into this feeling like I knew a decent bit already, but was excited to expand on it. Through this course I feel I more fully built out a skill set in front end web development. This class brought in far more practical use cases with a lot of hands on work that allowed me to put each of the lessons into practice. I have far greater understandings of syntax as well as how the different parts of web development like CSS and JavaScript communicate. I feel far more confident in this type of work.
The signature assignment, being the portfolio you are looking at, brings together all the elements of the semester's lessons. It makes use of HTML, CSS, JavaScript, and Bootstrap. This assignment has taught me how to effectively make use of all of these to create a cohesive website. It also assisted in teaching me immensely useful skills in how to find the needed resources. The ability to use documentation as well as resources like W3 schools. It helped teach me the best methods of troubleshooting issues. Tools like commenting out pieces to try and find the root cause or using border frames to better visualize what’s occurring. Being able to effectively find the problem is half the battle to solving the issue, and will be something I use often going forward.
This course I was able to vastly improve my knowledge and understanding of JavaScript. Easily one of the areas of front end web I had struggled the most previously. Being able to develop a mastery of it, proves massively useful in creating interactive responsive websites. Creating more fluid and responsive navigation or even being able to create specific triggers through JavaScript is a huge part of modern web design. I also believe the course has more generally expanded my ability to research and find answers. Often finding the answer requires being incredibly specific to the kinds of tags or desired outcomes for the language being used. It also helps to know more reliable sources like stack overflow and github to assist in solving problems or any questions I might have. I feel like these developments allow me to be more confident in all of front end web development.
I also learned A LOT about Metallica, but I don’t think I’m being graded on that.
